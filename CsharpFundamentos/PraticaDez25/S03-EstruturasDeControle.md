**Estruturas de Controle:** são usadas para controlar o fluxo de execução do código
 
**Estruturas condicionais:** permitem executar blocos de código com base em condições
    if: executa um bloco de código se a condição for verdadeira
    else: executa um bloco de código se a condição do if for falsa
    else if: permite verificar múltiplas condições sequencialmente
    switch: seleciona um bloco de código para executar com base no valor de uma expressão

**Estruturas de repetição:** permitem executar blocos de código repetidamente enquanto uma condição for verdadeira
    goto e label: permitem saltar para uma parte específica do código (uso geralmente desaconselhado)
    for: repete um bloco de código um número específico de vezes
    foreach: itera sobre elementos em uma coleção ou array
    while: repete um bloco de código enquanto a condição for verdadeira
    do...while: repete um bloco de código pelo menos uma vez e continua enquanto a condição for verdadeira

**Instruções de controle de loop:** alteram o fluxo dentro das estruturas de repetição
    break: sai imediatamente do loop mais próximo
    continue: pula para a próxima iteração do loop mais próximo
    return: sai de um método e opcionalmente retorna um valor

**Expressões condicionais (operador ternário):** permite atribuir valores com base em uma condição
    Sintaxe: condição ? valor_se_verdadeiro : valor_se_falso
