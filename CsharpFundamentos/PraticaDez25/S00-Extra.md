**Tópicos para se aprofundar**

-Classe Random: Geração de números aleatórios, métodos Next(), NextDouble(), NextBytes().
-Delegates e Eventos: Criação e uso de delegates, definição e manipulação de eventos.
-Programação Assíncrona: Uso de async e await, Task Parallel Library (TPL).
-Generics Avançados: Restrições de tipos genéricos, métodos genéricos.
-Algoritmo Determinístico vs Não Determinístico: é um algoritmo que produz o mesmo resultado para a mesma entrada, enquanto um algoritmo não determinístico pode produzir resultados diferentes para a mesma entrada.
-ThreadStatic: Uso de variáveis estáticas específicas para cada thread.
-Reflection: Inspeção de tipos em tempo de execução, obtenção de informações sobre assemblies
